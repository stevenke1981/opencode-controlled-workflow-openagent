# /start-work — Start Controlled Work

Use Odin as the lead.

1. Restate the user goal in one sentence.
2. Ask Athena for hidden assumptions if the task is ambiguous or large.
3. Ask Prometheus for a phased plan.
4. Convert the plan into todos with clear done conditions.
5. Start with Explore/Librarian/Merlin only when evidence is needed.
6. Do not ask the user unless a blocker would cause wrong direction, destructive action, or unsafe behavior.

Output:
- Goal
- Assumptions
- Todos
- First action
