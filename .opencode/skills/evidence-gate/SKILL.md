# Evidence Gate Skill

Use this before final output.

## Evidence required
- Changed files or generated artifacts.
- Commands run and their result.
- Relevant test/build/lint/manual validation.
- Known limitations or unverified parts.

## Completion language
Do not say "done" unless there is evidence.
Use:
- Completed and verified with ...
- Completed but not verified because ...
- Blocked by ...

## Minimal final template
```text
完成項目：
- ...

驗證：
- ...

剩餘風險：
- ...
```
